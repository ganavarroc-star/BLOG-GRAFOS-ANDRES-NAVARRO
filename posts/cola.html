Estructura de Datos Fundamental: First In, First Out

¬øQu√© es una Cola?

Una cola (queue) es una estructura de datos lineal que sigue el principio FIFO (First In, First Out). El primer elemento que entra es el primero en salir, exactamente como una fila de personas esperando en un banco o supermercado.

Caracter√≠sticas Principales

  • FIFO: Primer elemento insertado es el primero en ser extra√≠do
  • Operaci√≥n de inserci√≥n: Se realiza por el final (rear/back)
  • Operaci√≥n de extracci√≥n: Se realiza por el inicio (front/head)
  • Lineal: Los elementos se organizan en una secuencia
  • Din√°mica: Puede crecer o reducirse din√°micamente

Representaci√≥n Visual

üîÑ Operaciones Principales

1. Enqueue (Insertar)

A√±ade un elemento al final de la cola.

def enqueue(self, elemento):
    """Inserta un elemento al final"""
    self.elementos.append(elemento)

Complejidad: O(1)

2. Dequeue (Extraer)

Extrae y retorna el primer elemento de la cola.

def dequeue(self):
    """Extrae el primer elemento"""
    if not self.esta_vacia():
        return self.elementos.pop(0)
    return None

Complejidad: O(n) en listas, O(1) en colas con punteros

3. Front (Frente)

Retorna el primer elemento sin extraerlo.

def front(self):
    """Retorna el primer elemento sin eliminarlo"""
    if not self.esta_vacia():
        return self.elementos[0]
    return None

Complejidad: O(1)

4. Rear (Final)

Retorna el √∫ltimo elemento sin extraerlo.

def rear(self):
    """Retorna el √∫ltimo elemento"""
    if not self.esta_vacia():
        return self.elementos[-1]
    return None

Complejidad: O(1)

5. Is Empty (Est√° Vac√≠a)

Verifica si la cola est√° vac√≠a.

6. Size (Tama√±o)

Retorna el n√∫mero de elementos en la cola.

class Cola:
    """Implementaci√≥n de una Cola (Queue) FIFO"""
    
    def __init__(self):
        """Inicializa una cola vac√≠a"""
        self.elementos = []
    
    def enqueue(self, elemento):
        """A√±ade un elemento al final"""
        self.elementos.append(elemento)
    
    def dequeue(self):
        """Elimina y retorna el primer elemento"""
        if not self.esta_vacia():
            return self.elementos.pop(0)
        return None
    
    def esta_vacia(self):
        """Verifica si la cola est√° vac√≠a"""
        return len(self.elementos) == 0
    
    def tama√±o(self):
        """Retorna el n√∫mero de elementos"""
        return len(self.elementos)
    
    def front(self):
        """Retorna el primer elemento sin eliminarlo"""
        if not self.esta_vacia():
            return self.elementos[0]
        return None
    
    def rear(self):
        """Retorna el √∫ltimo elemento"""
        if not self.esta_vacia():
            return self.elementos[-1]
        return None
    
    def mostrar(self):
        """Muestra todos los elementos"""
        if self.esta_vacia():
            print("Cola vac√≠a: []")
        else:
            print(f"Cola: {self.elementos}")

# Ejemplo de uso

cola.enqueue("Cliente 1")
cola.enqueue("Cliente 2")
cola.enqueue("Cliente 3")

cola.mostrar()  # Cola: ['Cliente 1', 'Cliente 2', 'Cliente 3']

4. B√∫squeda BFS en Grafos

Las colas son esenciales para el algoritmo BFS:

def bfs(grafo, inicio):
    """B√∫squeda en Amplitud usando Cola"""
    visitados = set()
    cola = deque([inicio])
    visitados.add(inicio)
    
    while cola:
        nodo = cola.popleft()
        print(f"Visitando: {nodo}")
        
        for vecino in grafo[nodo]:
            if vecino not in visitados:
                visitados.add(vecino)
                cola.append(vecino)

üìä Cola con Prioridad

Variante de cola donde los elementos se extraen seg√∫n su prioridad, no por orden de llegada:

import heapq

class ColaConPrioridad:
    """Cola donde se atienden primero elementos de mayor prioridad"""
    
    def __init__(self):
        self.elementos = []
    
    def enqueue(self, elemento, prioridad=0):
        """Menor n√∫mero = mayor prioridad"""
        heapq.heappush(self.elementos, (prioridad, elemento))
    
    def dequeue(self):
        """Extrae el elemento de mayor prioridad"""
        if self.elementos:
            return heapq.heappop(self.elementos)[1]
        return None

# Ejemplo: Urgencias en hospital
urgencias = ColaConPrioridad()
urgencias.enqueue("Herida leve", 3)
urgencias.enqueue("Infarto", 0)
urgencias.enqueue("Fractura", 2)

# Se atienden en orden de urgencia, no de llegada
print(urgencias.dequeue())  # "Infarto" (prioridad 0)
print(urgencias.dequeue())  # "Fractura" (prioridad 2)
print(urgencias.dequeue())  # "Herida leve" (prioridad 3)
